# Note Taking Application

This project is a Note Taking Application built using React.js. It allows users to create, view, and delete notes in a user-friendly and interactive manner. The application consists of components such as Header, Footer, Note, and CreateArea, providing a seamless user experience.

## Features

- **Add Note**: Users can create and add new notes to the application.
- **Delete Note**: Users can delete individual notes based on their preference.
- **Dynamic Rendering**: Notes are dynamically rendered on the screen, allowing for a smooth and efficient user experience.
- **Component-based Architecture**: The application is structured using reusable components, ensuring a clean and maintainable codebase.
- **React Hooks**: React's useState hook is utilized to manage and update note data in a dynamic manner.

## Usage

- Upon launching the application, users will see a header and a CreateArea component at the top.
- Users can input a note title and content in the CreateArea component.
- Clicking the "Add" button will add the note to the list of notes below.
- Each note displayed includes a title, content, and a delete button.
- Users can delete a note by clicking the corresponding delete button.
- The Header and Footer components are displayed at the top and bottom of the application, respectively.
